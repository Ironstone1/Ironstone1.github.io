// --- What the commands mean ---
crosshair [0, 1] - hide / show the crosshair


cl_crosshairalpha [0, onwards+] - changes how transparent the crosshair is

math_counter
on func_button press --> add 1 to counter --> cl_crosshairalpha {counter value}


cl_crosshaircolor - changes the colour of the cross hair:
0 - red
1 - green
2 - yellow
3 - dark blue
4 - light blue
5 - custom RGB color:
	- custom colors:
		- cl_crosshaircolor 5;
		- cl_crosshaircolor_r [0 to 255]
		- cl_crosshaircolor_g [0 to 255]
		- cl_crosshaircolor_b [0 to 255]
		
for example: 		
cl_crosshaircolor 5;
cl_crosshaircolor_r 0;
cl_crosshaircolor_g 191;
cl_crosshaircolor_b 255;


cl_crosshairdot
0 - no dot
1 - dot


cl_crosshairgap [0, onwards] - changes the number of pixels between the crosshair dot and outer edges.


cl_crosshairsize [0, onwards] - changes the size of the corshair, how big is it displayed on screen


cl_crosshairstyle [0, to 6] - changes the style of the corsshair


cl_crosshairusealpha 
0 - crosshair looks a little more faded
1 - crosshair looks brighter


cl_crosshairthickness [0, to onwards] - changes how thick the crosshair lines are


cl_crosshair_drawoutline
0 - draws no outline around the crosshair
1 - draws a outline around the crosshair
