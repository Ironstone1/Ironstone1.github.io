SYSTEM32 APPLICATIONS:

The following Menu contains 116 items,
The menu contains a variation on “exe” and “msc” files. 
All of which have been taken from the %windir%\system32 folder, 
of a Microsoft Windows 7 Pro Service Pack 1 Operating System.
Beware that depending on your version of windows some of these features many not work.
The idea of this Menu is to give you direct access, to a detailed list of core operating system files. use this Menu at your own risk! some of these files can cause serious 
or devestating damage, as they are all necessary operating system files.
