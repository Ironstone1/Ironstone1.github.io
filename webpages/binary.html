How to write letters in Binary:

  1. First come up with a string, for example i will be using the string "hello".
  2. draw out a 8 by len(string) table, len(string) meaning the length of the string you are writing in binary, for example "hello" is length 5 so my table would be 8x5 table. In binary (base 2) the digits across the top can be worked out by calculating 2**i, for i being evey integer from 0 to 8, and the '2' representing base 2 'binary'.
  3. Next, to write lowercase letters, start by writing the number 96 in binary (to work out upercase letters just start with 64 instead of 96). to write a number in binary add the colums acrors the top up to workout the number. for example to get the number 96 you add 64 + 32, so put a '1' in both those colums and fill the reset of the colums with 0's. this calculates the binary value "01100000".

    Next work out the order of the letter in the alphabet, for example 'a' is the first letter in the alphabet so will have order 1. if we look back at the example the first character of the string is the letters 'h', 'h' is the 8th letter in the alphabet. so to work out the binary value for the letter 'h' we calculate 96 + 8, 8 being how far the letter is in the alphabet. this gives us the integer 104. now if we add the colums up accross the top of the table to calculate the number 104, we are given the binary value "01101000", so 'h' in binary is '01101000'.
