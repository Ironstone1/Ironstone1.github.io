How to use:
[olist]
[*]Go to your 'C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\maps\workshop\2048065424' folder
[*]Copy the file 'cold_pizza_achiv_2.bsp'
[*]Then paste the file in the 'C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\maps' folder
[*]Finally open CSGO, open the console and type "map cold_pizza_achiv_2 demolition".
[*]Once the map loads and you are asked to select a team, place a heavy object on your keyboard space bar.
[/olist]
