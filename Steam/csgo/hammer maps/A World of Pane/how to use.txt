A map to help get the "A World of Pane", "Shoot out 14 windows in a single round on Office" achievement.

[img]http://i.imgur.com/IFcoJhCg.png[/img]

[b]Important! Please read the how to use instructions below, else this map wont work[/b]

[img]http://i.imgur.com/IFcoJhCg.png[/img]

How to use:
[olist]
[*]go to your 'C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\maps\workshop\1907565555' folder
[*]copy the file 'cs_office.bsp'
[*]then paste the file in the 'C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\maps' folder
[*]If there is already a file called cs_office in your maps folder, save a copy of it to your desktop or somewhere else safe so you have a backup of the original file. 
[*]Then delete the original cs_office.bsp file in your maps folder, and paste the new cs_office.bsp file that you copied.
[*]Finally open CSGO, open the console and type "map cs_office casual", once the map loads just wait until you get the achievement.
[/olist]
